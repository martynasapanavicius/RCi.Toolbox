# Release notes

- Version 2.0.0 - 2026-04-01
  - Bumped to target framework to `net10.0`
  - Removed `DateTimePrecise` and `DateTimeOffsetPrecise`
  - Added `SleepExtensions`
  - Added `Box<T>`, `SyncBox<T>` and `SyncBoxDeferred<T>`
  - Added `AtomicGate`
  - Updated `JobQueue` (fixed rare race conditions)
  - Added `CoalescingWorker`

- Version 1.0.0 - 2025-11-03
  - Target framework `net8.0`
  - Added `DateTimePrecise` and `DateTimeOffsetPrecise`
  - Added `ValueStopwatch`
  - Added `JobQueue`
